Usage: noctalia msg <command> [args]

Commands:
  bar-auto-hide-set <on|off|true|false|1|0> [bar-name] [monitor-selector]           Set auto-hide state for a bar
  bar-hide [bar-name] [monitor-selector]                                            Hide one or all bars and release their layout gaps
  bar-layer-set <top|overlay> [bar-name] [monitor-selector]                         Set one or all bar layers
  bar-reserve-toggle [bar-name] [monitor-selector]                                  Toggle reserve space for one or all bars
  bar-show [bar-name] [monitor-selector]                                            Show one or all bars
  bar-toggle [bar-name] [monitor-selector]                                          Toggle visibility for one or all bars
  bluetooth-disable                                                                 Disable Bluetooth
  bluetooth-enable                                                                  Enable Bluetooth
  bluetooth-status                                                                  Print Bluetooth state
  bluetooth-toggle                                                                  Toggle Bluetooth
  brightness-down [current|*|all|monitor-selector] [step]                           Decrease brightness (defaults to current monitor)
  brightness-list-backlight-devices                                                 List available sysfs backlight device names
  brightness-osd <value>                                                            Show brightness OSD without changing brightness
  brightness-set <value> | brightness-set <current|*|all|monitor-selector> <value>  Set brightness (defaults to current monitor)
  brightness-up [current|*|all|monitor-selector] [step]                             Increase brightness (defaults to current monitor)
  caffeine-disable                                                                  Disable caffeine (idle inhibitor)
  caffeine-enable                                                                   Enable caffeine (idle inhibitor)
  caffeine-toggle                                                                   Toggle caffeine (idle inhibitor)
  clipboard-clear                                                                   Clear clipboard history
  color-scheme-get                                                                  Print active color scheme: <source> <name> (source is builtin, wallpaper, community, or custom)
  color-scheme-set <source> <name>                                                  Set palette source and selection in settings.toml (builtin name, wallpaper generator scheme, community id, or custom scheme folder name)
  config-reload                                                                     Reload the config file
  desktop-widgets-edit                                                              Open the desktop widgets editor
  desktop-widgets-exit                                                              Close the desktop widgets editor
  desktop-widgets-hide                                                              Hide desktop widgets now (runtime only; does not change the saved setting)
  desktop-widgets-show                                                              Show desktop widgets now (runtime only; does not change the saved setting)
  desktop-widgets-toggle                                                            Toggle desktop widgets visibility (runtime only; does not change the saved setting)
  desktop-widgets-toggle-edit                                                       Toggle desktop widgets edit mode
  dock-hide                                                                         Hide the dock (persists override)
  dock-reload                                                                       Reload dock configuration
  dock-show                                                                         Show the dock (persists override)
  dock-toggle                                                                       Toggle dock visibility (persists override)
  dpms-off                                                                          Turn monitors off
  dpms-on                                                                           Turn monitors on
  effects-profile-set <output|input> <profile>                                      Set the EasyEffects output or input profile
  greeter-sync                                                                      Sync wallpaper, colors, and monitor layout to Noctalia Greeter
  lockscreen-widgets-edit                                                           Open the lockscreen widgets editor
  lockscreen-widgets-exit                                                           Close the lockscreen widgets editor
  lockscreen-widgets-toggle-edit                                                    Toggle lockscreen widgets edit mode
  log-level-set <debug|info|warn|error>                                             Set the console log level
  log-level-status                                                                  Print the current console log level
  media <next|previous|toggle|play|pause|stop|next-player|previous-player>          Control active media playback
  mic-mute                                                                          Toggle microphone mute
  mic-volume-down [step]                                                            Decrease microphone volume
  mic-volume-set <value>                                                            Set microphone volume
  mic-volume-up [step]                                                              Increase microphone volume
  nightlight-disable                                                                Disable night light schedule
  nightlight-enable                                                                 Enable night light schedule
  nightlight-force-toggle                                                           Toggle forced night light mode
  nightlight-toggle                                                                 Toggle night light schedule
  notification-clear-active                                                         Dismiss all currently active notifications
  notification-clear-history                                                        Clear notification history
  notification-dnd-set <on|off|true|false|1|0>                                      Set notification Do Not Disturb state
  notification-dnd-status                                                           Print notification Do Not Disturb state
  notification-dnd-toggle                                                           Toggle notification Do Not Disturb state
  notification-invoke-latest                                                        Invoke the default action of the most recent active notification
  notification-show <summary [-- body]|json>                                        Show an internal Noctalia notification
  panel-close [id]                                                                  Close the active panel, or close the named panel if it is active
  panel-open <id> [context]                                                         Open a panel by id, optionally with context (e.g. launcher /emo, control-center audio)
  panel-toggle <id> [context]                                                       Toggle a panel by id, optionally with context (e.g. launcher /emo, control-center audio)
  plugin <author/plugin:entry> <target[:bar-name]> <event> [payload]                Dispatch an event to a plugin entry
  plugins <list|enable|disable|update|source> ...                                   Manage plugins and sources (list/enable/disable/update, source list/add/remove)
  power-cycle                                                                       Switch to the next power profile in UPower's ordered list (wraps)
  power-set <profile>                                                               Set the UPower power profile (e.g. performance, balanced, power-saver)
  screenshot-fullscreen [pick|monitor|all]                                          Capture the focused monitor by default, pick interactively with pick, or all outputs with all
  screenshot-region                                                                 Start an interactive region screenshot
  session <lock|suspend|lock-and-suspend|logout|reboot|shutdown>                    Run a built-in session action
  settings-close                                                                    Close the settings window
  settings-open [context]                                                           Open the settings window, or focus it if already open, optionally at a specific section
  settings-toggle [context]                                                         Toggle the settings window, optionally at a specific section
  status                                                                            Print current state as JSON
  templates-apply                                                                   Apply configured theme templates for the current palette
  theme-mode-get                                                                    Print the current resolved theme mode
  theme-mode-set <dark|light|auto>                                                  Set theme mode and persist to settings.toml
  theme-mode-toggle                                                                 Toggle theme mode between dark and light
  volume-down [step]                                                                Decrease speaker volume
  volume-mute                                                                       Toggle speaker mute
  volume-set <value>                                                                Set speaker volume
  volume-up [step]                                                                  Increase speaker volume
  wallpaper-get [<connector>]                                                       Print default wallpaper path, or effective path for an output
  wallpaper-next [<connector>]                                                      Switch to the next wallpaper immediately
  wallpaper-previous [<connector>]                                                  Switch to the previous wallpaper immediately
  wallpaper-random [<connector>]                                                    Switch to a random wallpaper immediately
  wallpaper-set [<connector>] <path>                                                Set wallpaper for all or a specific output (persisted)
  wifi-disable                                                                      Disable Wi-Fi
  wifi-enable                                                                       Enable Wi-Fi
  wifi-status                                                                       Print Wi-Fi state
  wifi-toggle                                                                       Toggle Wi-Fi
  window-switcher [close]                                                           Open or close the window switcher overlay
  workspace-alert-add <workspace>                                                   Add a workspace alert (by number, name, or id)
  workspace-alert-add-window <window-id>                                            Add a workspace alert for a window
  workspace-alert-clear <workspace>                                                 Clear a workspace alert
  workspace-alert-clear-all                                                         Clear all workspace alerts
  workspace-alert-status                                                            Print workspace alerts
